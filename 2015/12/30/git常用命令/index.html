git常用命令

发表于 2015-12-30   |   分类于 git

git的一些自己常使用的简单命令

设置git的user name和email：

$ git config --global user.name "xiaoming"
$ git config --global user.email "xiaoming@gmail.com"

在本机生成秘钥：

$ ssh-keygen -t rsa -C "xiaoming@gmail.com"

按3个回车，这样密码设置的就是空。
ssh秘钥在.ssh目录下，可以查看是否已经有了秘钥。

在github上建立好仓库，并且设置完公钥后，对其进行确认的命令：

ssh -T git@github.com

显示如下则正确：

Warning: Permanently added 'github.com,207.97.227.239' (RSA) to the list of known hosts.
　　Hi findingsea! You've successfully authenticated, but GitHub does not provide shell access.

仓库初始化：

git init

生成快照并存入项目索引：

git add

项目索引提交：

git commit

推送本地更新到远程：

git push origin master

更新远程到本地：

git pull origin master
